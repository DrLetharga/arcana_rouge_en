# Event CG Guidelines

Event CGs represent key narrative moments and character expressions.

- **Location:** `source/g_event/`
- **Tag Match:** `[evcg storage="ev12b"]` → `ev12b.png`
- **Purpose:** Highlight impactful, emotional, or climactic scenes.
- **Best Practices:**
  - Use image content to infer emotional tone.
  - Prioritize visual emotion over literal text.
  - Adjust descriptions for what’s visible (position, gesture, state of dress, etc.).

**Special Notes:**
- Use images to interpret ambiguous dialogue.
- Reconcile visual expressions with vocal audio to resolve tone conflicts.
- If characters appear in erotic scenes, assess pose/dynamics before assuming dominance/submission.
