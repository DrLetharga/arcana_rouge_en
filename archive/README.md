KiriKiri Translation Project

This is a repository for the English translation of a Japanese visual novel built using the KiriKiri engine. The project includes translated script files, tools for encoding and macro conversion, and documentation for workflow and style.

Structure

source/: Unmodified Japanese source files (scripts, images, audio).

translation/: Translated files, organized by type.

tools/: Helper scripts for file renaming, encoding conversion, and macro cleanup.

doc/: Documentation on the game's code and internal logic.

archive/: Backup of abandoned or previous versions.

Goals

Faithfully translate the script while preserving tone and intent.

Ensure compatibility with KiriKiri Z.

Retain and preserve formatting macros where appropriate.
